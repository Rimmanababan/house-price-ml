# House Price Machine Learning

Project Machine Learning untuk memprediksi harga rumah menggunakan dataset housing.

## Dataset
Project ini menggunakan dua dataset:
- kc_house_data.csv
- clean.csv

## Tahapan Project

### 1. Feature Engineering
Dilakukan di notebook:
notebooks/feature_engineering.ipynb

Proses yang dilakukan:
- Membersihkan data
- Menghapus kolom yang tidak diperlukan
- Membuat fitur baru

### 2. Model Building
Dilakukan di notebook:
notebooks/build_model.ipynb

Model yang digunakan:
- XGBoost Regressor

Evaluasi model menggunakan:
- R2 Score

## Tools
- Python
- Pandas
- Scikit-Learn
- XGBoost
- Jupyter Notebook

## Tujuan
Membuat model Machine Learning dengan akurasi minimal R² ≥ 85% tanpa menggunakan Deep Learning.
